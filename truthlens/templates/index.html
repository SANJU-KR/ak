Process

From raw text to verdict in milliseconds.

Every headline passes through a complete NLP pipeline before the machine learning model issues its credibility score.

01
✍️

Text Normalization

Lowercase conversion, URL removal, number replacement, punctuation stripping, and whitespace normalization prepare the raw text for processing.

Preprocessing
02
✂️

NLP Feature Extraction

Porter stemming reduces words to roots, stopwords are removed, and 1–3-gram token sequences plus character n-grams (3–5) are generated for richer signal.

NLP Pipeline
03
📊

TF-IDF Vectorization

Each token is weighted by its local frequency (sublinear TF) multiplied by its global rarity (IDF), producing a sparse 3,775-dimensional feature vector.

Representation
04
💬

Sentiment & Style Analysis

A custom lexicon scores positive/negative polarity, intensifiers, hedge words, and 17 linguistic features including Flesch readability, caps ratio, and type-token ratio.

Feature Engineering
05
🤖

ML Classification

The combined feature vector is passed to a trained Logistic Regression model (best of 4 trained models) which computes a sigmoid probability over the full feature space.

Machine Learning
06
🎯

Credibility Verdict

The model outputs a credibility score from 0–100%, a FAKE/REAL classification, token-level explanations, sentiment breakdown, and detected red flags.

Output
Technology

🌱
Porter Stemmer
Suffix-stripping algorithm reduces inflected words to their root form. "researchers" → "research", "running" → "run". Improves generalization by collapsing morphological variants.
Morphological NLP
📈
TF-IDF (Sublinear)
Term Frequency-Inverse Document Frequency weights each token by (1 + log tf) × idf, balancing common and rare terms. The word and character vocabularies together yield 3,775 features.
Statistical NLP
🔗
N-Gram Extraction (1–3)
Unigrams, bigrams, and trigrams capture multi-word patterns. Phrases like "big pharma", "one simple trick", and "government secretly" become single features with strong model weight.
Sequence Modeling
🔤
Character N-Grams (3–5)
Sub-word character sequences (e.g. "SHOC", "!!!") catch ALL-CAPS writing, unusual punctuation, and spelling tricks that evade word-level tokenization. 3,000 character n-gram features.
Sub-Word Features
💡
Sentiment Analysis
Custom lexicon scores positive/negative word ratio, intensifiers ("absolutely", "completely"), hedge words ("reportedly", "allegedly"), and polarity. 6 sentiment dimensions.
Lexicon-Based NLP
📐
17 Linguistic Features
Flesch readability score, type-token ratio, syllable density, sentence length, capitals ratio, punctuation density, named entity count, fake/real pattern matches, and more.
